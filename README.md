# HalalLens

HalalLens is a web-based AI-powered application that helps users analyze food products by scanning ingredient labels.

## Features
- Image upload and OCR scanning
- Halal / Non-Halal / Uncertain classification
- Ingredient breakdown with explanations
- Scan history tracking
- Multi-language support
- Dark mode UI

## Tech Stack
- React + Vite
- FastAPI (backend)
- OCR / AI analysis
- Tailwind / CSS

## How to run locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Note
Backend must be running separately for full functionality.
